# 🔫 MONSTER SHOOTER - Мобільна гра-шутер

## 📝 Опис
Динамічна мобільна гра-шутер на HTML5, оптимізована для touch-управління. Вибирайте персонажа, стріляйте по монстрах, проходьте хвилі та покращуйте свою зброю!

## 🎮 Особливості

### ✨ Основні механіки:
- **5 унікальних персонажів**: Tito, Smartik, Junior, Fil, Milka
- **Touch-управління**: Тримайте і водіть пальцем для стрільби
- **Система хвиль**: Нескінченна прогресія з зростаючою складністю
- **Різні типи ворогів**: Від слабких на перших хвилях до босів після 10-ї
- **Візуальні HP бари**: Кожен ворог має індикатор здоров'я

### 🏪 Система покращень:
- **Урон**: Збільшення пошкоджень від пострілів
- **Швидкість стрільби**: Зменшення затримки між пострілами  
- **Здоров'я**: Збільшення максимального HP

### 💾 Збереження прогресу:
- Рекорд (найвища хвиля) зберігається НАЗАВЖДИ
- Куплені покращення не скидаються
- Зароблені монети зберігаються між сесіями

## 📱 Установка

### Варіант 1: Прямий запуск
1. Розмістіть всі файли в одній папці:
   - index.html
   - style.css
   - game.js
   - Tito.png, Smartik.png, Junior.png, Fil.png, Milka.png

2. Відкрийте `index.html` у браузері

### Варіант 2: Локальний сервер
```bash
# За допомогою Python 3
python -m http.server 8000

# Або за допомогою Node.js (npx http-server)
npx http-server -p 8000

# Потім відкрийте: http://localhost:8000
```

## 🎯 Як грати

1. **Вибір персонажа**: Натисніть на одного з 5 персонажів
2. **Стрільба**: Тримайте палець на екрані і водіть - автомат стріляє автоматично
3. **Мета**: Знищуйте монстрів, збирайте монети
4. **Прогрес**: Проходьте хвилі, купуйте покращення
5. **Виживання**: Не дайте монстрам знизити ваше HP до 0!

## 🌊 Система хвиль

- **Хвилі 1-3**: Слабкі зелені монстри (40 HP, повільні)
- **Хвилі 4-7**: Середні жовті монстри (80 HP, швидкі)
- **Хвилі 8-9**: Сильні помаранчеві монстри (150 HP, дуже швидкі)
- **Хвиля 10+**: Боси (250+ HP, максимальна швидкість і урон)

З кожною хвилею з'являється більше ворогів!

## 🎨 Зображення персонажів

Якщо у вас немає власних зображень персонажів:
1. Відкрийте `create_placeholders.html` у браузері
2. Автоматично завантажаться 5 placeholder зображень
3. Або додайте власні PNG файли з іменами:
   - Tito.png
   - Smartik.png
   - Junior.png
   - Fil.png
   - Milka.png

**Рекомендовані розміри**: 200x200 пікселів або більше

## 🔧 Технічні деталі

- **Canvas rendering**: 60 FPS анімація
- **LocalStorage**: Збереження даних
- **Touch events**: Оптимізоване управління
- **Responsive**: Адаптується до будь-якого екрану
- **Particle effects**: Візуальні ефекти попадань

## 📊 Система монет та покращень

### Нагороди за ворогів:
- Хвилі 1-3: 10 монет
- Хвилі 4-7: 20 монет  
- Хвилі 8-9: 35 монет
- Хвиля 10+: 50+ монет

### Ціни покращень:
- **Урон**: 50 × рівень
- **Швидкість**: 75 × рівень
- **Здоров'я**: 100 × рівень

## 🎮 Керування

### Мобільні пристрої:
- **Touch Start**: Почати стрільбу
- **Touch Move**: Змінити напрямок
- **Touch End**: Зупинити стрільбу

### ПК (для тестування):
- **Mouse Down**: Почати стрільбу
- **Mouse Move**: Змінити напрямок  
- **Mouse Up**: Зупинити стрільбу

### Інтерфейс:
- **⏸ Кнопка**: Пауза
- **Меню паузи**: Продовжити / Магазин / Рестарт / Вихід

## 🐛 Усунення проблем

### Зображення не завантажуються?
- Переконайтеся, що PNG файли в тій же папці
- Перевірте правильність імен файлів (регістр важливий!)
- Використайте `create_placeholders.html` для генерації тестових зображень

### Гра не працює на мобільному?
- Переконайтеся, що браузер підтримує Canvas
- Спробуйте Chrome або Safari (останні версії)
- Дозвольте JavaScript у налаштуваннях браузера

### Прогрес не зберігається?
- Перевірте, чи дозволені cookies/LocalStorage
- Не використовуйте режим інкогніто
- Очистіть кеш та спробуйте знову
